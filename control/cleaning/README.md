# Cleaning Scripts

Scripts in this directory remove intermediate data or outputs.

## Descriptions

- `clean_nfs_run.sh <run_name>`
  Removes NFS data for a specific run ID only.

- `clean_nfs_all_runs.sh`
  Removes **all** run directories from NFS.

- `clean_minio_run.sh <run_name>`
  Deletes MinIO objects associated with a specific run.
